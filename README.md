fastofd
⚡ 一个专注于高效解析政府采购投标 OFD 文件并支持多模态分析的工具

fastofd 是基于 easyOFD 进行深度优化与场景拓展的开源库。在保留其核心解析能力的基础上，我们针对 政府采购与招投标领域 的实际需求进行了多项关键改进，显著提升了对复杂 OFD 文件的兼容性与处理效率。

本项目主要聚焦于将 OFD 文档快速、稳定地转换为 PDF 或 JPG 格式，便于后续进行多模态大模型分析或 OCR 识别，助力自动化评标、信息抽取等智能化场景。

🌟 主要改进
✅ 增强解析能力：优化了对政府采购及招投标场景中各类复杂 OFD 文件的支持，能够解析更多非标准结构和边缘情况。
🧩 简化字体配置：移除了原项目中繁琐的字体依赖配置，默认使用内置统一体渲染策略，降低部署复杂度，提升跨平台一致性。
🚀 高性能输出：专注于 OFD 到 PDF/JPG 的高效转换，输出结果更适配下游的图像识别与多模态模型输入需求。
📦 轻量化设计：精简不必要的模块，更适合集成到自动化处理流水线中。
🎯 典型应用场景
投标文件自动解析与结构化
结合 OCR 提取文本信息（如供应商名称、金额、工期等）
集成至多模态 AI 模型，实现智能审查与比对
批量生成可读性强的 PDF 或图片格式归档文件
